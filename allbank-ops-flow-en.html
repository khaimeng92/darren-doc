AllBank Account Linking
How players link and unlink their AllBank account via Registration Link
Player Web UI
FPMS Player System
PMS Payment System
AllBank External Bank
1
Player
Player clicks "Link AllBank Account"
The system requests a unique registration link to be generated
2
FPMS → PMS
A unique AllBank registration link is generated
The Player System requests a registration URL from the Payment System, which fetches it from AllBank
3
Player
Player is redirected to AllBank to complete registration
Player logs in and grants consent on AllBank's official page — fully handled by AllBank
4
AllBank
AllBank notifies the system of the result
Once the player completes registration, AllBank automatically sends back the player's unique Link ID
5
PMS → FPMS
Link ID is saved to the player's profile
The Payment System forwards the result to the Player System, which stores the AllBank Link ID against the player's account
✓
Player
Linking complete — account shows as Linked
Player can now use AllBank for deposits and withdrawals
1
Player
Player clicks "Unlink AllBank Account"
The system initiates an unlink request
2
FPMS → PMS
Player System passes the Link ID to Payment System
FPMS retrieves the player's AllBank Link ID and sends it to PMS to process the revocation
3
PMS → AllBank
Payment System requests AllBank to revoke the binding
AllBank removes the player's authorisation and confirms the revocation
4
PMS → FPMS
Result is passed back to the Player System
FPMS updates the player's profile and removes the AllBank Link ID
✓
Player
Unlinking complete — account shows as Not Linked
AllBank payment methods are disabled. Player may re-link at any time
