AllBank 账户绑定流程
玩家通过注册链接（RTL）绑定 / 解绑 AllBank 账户
FE 前端页面
FPMS 玩家信息系统
PMS 支付系统
AllBank 银行（外部）
1
玩家 / FE
玩家点击「绑定 AllBank 账户」
系统请求生成一条专属注册链接
2
FPMS → PMS
生成 AllBank 注册链接
FPMS 通知支付系统，支付系统向 AllBank 申请一条专属注册 URL
3
玩家 / FE
跳转到 AllBank 页面完成注册
玩家在 AllBank 官方页面登录并授权绑定，操作完全由 AllBank 处理
4
AllBank
AllBank 通知绑定结果
玩家完成注册后，AllBank 自动回调通知支付系统，返回玩家绑定 ID（playerLinkId）
5
PMS → FPMS
支付系统转发绑定结果给玩家系统
支付系统验证回调，将 AllBank 绑定 ID 传给 FPMS，由 FPMS 保存至玩家档案
✓
玩家 / FE
绑定完成，页面显示「已绑定」
玩家可正常使用 AllBank 进行存取款
1
玩家 / FE
玩家点击「解绑 AllBank 账户」
系统发起解绑请求
2
FPMS → PMS
玩家系统通知支付系统解绑
FPMS 获取该玩家的 AllBank 绑定 ID，传给支付系统处理
3
PMS → AllBank
支付系统向 AllBank 发起解绑
AllBank 撤销该玩家的绑定授权，返回确认结果
4
PMS → FPMS
解绑结果回传玩家系统
FPMS 更新玩家档案，移除 AllBank 绑定记录
✓
玩家 / FE
解绑完成，页面显示「未绑定」
AllBank 相关支付功能暂停，玩家可重新绑定
